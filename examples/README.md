# 技能示例 (Examples)

这里存放了一些基于 `ReactiveSkill V2` 架构编写的 JSON 脚本示例，可供外部开发者和极客玩家学习与参考。

## 示例索引

1. **[01_自动刷视频.json](./01_自动刷视频.json)**
   - **描述**：一个极简的演示脚本。展示了如何利用 `constraints` 中的 `cooldownMs` 和 `executionMode: "REPEAT"` 来实现无限次的安全循环上滑操作。
   - **特点**：无代码循环控制，通过原生的 Goal 机制实现沉浸式浏览。

2. **[02_智能抖音V2.json](./02_智能抖音V2.json)**
   - **描述**：一个高阶演示脚本。充分体现了 V2 架构的“纯本地响应式”特征。
   - **特点**：
     - **全局拦截 (Interrupts)**：利用独立线程实时拦截“青少年模式”和“版本更新”弹窗。
     - **语义点击 (Semantic)**：直接通过描述文字定位按钮，无需绝对坐标。
     - **相对滑动**：演示了如何在一个特定的列表控件（如评论区）内进行安全的局部百分比滑动。

## 如何运行
确保您已经启动了本地的 `kuaiyou-mcp-server`，并将 AI 助理与您的 Android 设备连接成功。你可以直接让 AI 读取这些脚本内容并推送到手机执行，或者参考这些代码编写自己的自动化任务。
